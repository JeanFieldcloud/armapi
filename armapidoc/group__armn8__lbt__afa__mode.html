Enable/Disable and configure the Lbt&Afa mode.

Warning
If you have the WOR mode enable (in ARMN8_WOR_CS see armN8SetWorMode()) and the mode is not ARMN8_LBTAFA_DISABLE the rssiLevel is the same register in ARM, so if change this value with this function, the rssiLevel value of the WOR mode is also modified.
Parameters
armPointer to your ARM structure.
modeThe value available:
  • ARMN8_LBTAFA_DISABLE To disable the LBT&AFA mode.
  • ARMN8_LBTAFA_LBT To enable Listen Before Talk mode. Use this to emit on the radio if the rssi is less than rssiLevel.
  • ARMN8_LBTAFA_AFA To enable Adaptive Frequency Agility mode. Use this to receive on one channel of the two (channel in armN8SetRadio() and channel2).
  • ARMN8_LBTAFA_LBTAFA To enable Listen Before Talk and adaptive Frequency Agility, listen on the tow channels (channel in armN8SetRadio() and channel2) and talk on the free channels.
rssiLevelThis parameter in dBm is used by ARMN8_LBTAFA_AFA and ARMN8_LBTAFA_LBTAFA (the default value is -95) for define the level rssi who say if the channels is busy or not.
nSamplesThis is the number of rssi sample (the default value is 240).
channel2The secondary channel, the primary channel (and the other radio parameters) must be set with armN8SetRadio() function.
  • The range possible is of 1 to 559.
  • The function, return the error ARMN8_ERR_PARAM_OUT_OF_RANGE if this value is out of rang.
Returns
Error available:
  • ARMN8_ERR_NONE If successful.
  • ARMN8_ERR_PARAM_OUT_OF_RANGE If one of the parameters is out of rang.
  • ARMN8_ERR_PARAM_INCOMPATIBLE The parameter channel2 of this function is link with the parameters of the function armN8SetRadio(). Is very important to chose the correct value. For more information, please consulate the main radio parameter page and the documentation of armN8SetRadio() function.
Note
To configured the other radio parameters please call armN8SetRadio() function.
You need to call armN8UpdateConfig() for update the parameters in you ARM.

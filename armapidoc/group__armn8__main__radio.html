Get the main radio configuration.

Please, for more information about parameter consult the documentation of the armN8SetRadio() function.

Note
If you don't went get a parameter you can give the NULL value.
Parameters
armPointer to your ARM structure.
channelIf the pointer is valid, the channel is set.
baudIf the pointer is valid, the baud is set. If ARMN8_BAUDRATE_NONE is set, there is probably a error.
powerIf the pointer is valid, the power is set. If the power is no particularity specified, the ARMN8_POWER_AUTO value is set.

Parameters
armPointer to your ARM structure.
channelThe main radio channel.
  • The range possible is of 1 to 559.
  • The function, return the error ARMN8_ERR_PARAM_OUT_OF_RANGE if this value is out of rang.
baudThe radio baudrate.
  • The function, return the error ARMN8_ERR_PARAM_OUT_OF_RANGE if you give a another value defined below.
  • Is possible to give a integer value. For example for ARMN8_BAUDRATE_19200 you can directly give 19200.
  • The possible value is:
    • ARMN8_BAUDRATE_1200 to 1200bps.
    • ARMN8_BAUDRATE_2400 to 2400bps.
    • ARMN8_BAUDRATE_4800 to 4800bps.
    • ARMN8_BAUDRATE_9600 to 9600bps.
    • ARMN8_BAUDRATE_19200 to 19200bps.
    • ARMN8_BAUDRATE_38400 to 38400bps.
    • ARMN8_BAUDRATE_57600 to 57600bps.
    • ARMN8_BAUDRATE_115200 to 115200bps.
powerThe radio output power.
  • If you have a ARM in LD 'Long Distence' version the max value is 27 for 27dBm.
  • If you have a ARM in LP 'Low Power' version the max value is 14 for 14dBm.
  • The min value is -18 for -18dBm.
  • You can passe the ARMN8_POWER_AUTO value to set the max power according from the channel and baud parameter. In this case, you can choose the channel and baud value without worry the power value.
Returns
Error available:
  • ARMN8_ERR_NONE If successful.
  • ARMN8_ERR_PARAM_OUT_OF_RANGE If one of the parameters is out of rang.
  • ARMN8_ERR_PARAM_INCOMPATIBLE The parameters of this function is link between them. Is very important to chose the correct value. For more information, please consulate the main radio parameter page.
Note
You need to call armN8UpdateConfig() to update the parameters in you ARM.

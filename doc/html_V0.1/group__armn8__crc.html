Detailed Description

In the case or the CRC is enable:

  • The ARM module compute the CRC and send on the radio the frame with the CRC.
  • If the ARM module receive the radio frame with the CRC (the emitter must have the CRC enable), the ARM compute the CRC and throw the frame if the CRCs don't match.

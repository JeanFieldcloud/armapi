armN8Error_t armN8UpdateConfig ( armN8_t *  arm)

Update the configuration in ARM.

None of parameters/modes previously configured, is no set in your ARM. This function send/set the necessary registers to apply the previously configuration and reconfigure the port if necessary.

Parameters
armPointer to your ARM structure.
Returns
Error available:
  • ARMN8_ERR_NONE If success.
  • ARMN8_ERR_PORT_CONFIG If the port can't is configured.
  • ARMN8_ERR_PORT_WRITE_READ If can't write or read through the port.

  • ARMN8_ERR_ARM_BACK_AT If can't back AT commend.

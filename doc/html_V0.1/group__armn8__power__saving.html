armN8Error_t armN8EnableWakeUpUart ( armN8_t *  arm,
bool  enable 
)

Enable/Disable the Wake Up Uart.

Warning
The Wake Up Uart don't supporter the serially baudrate upper than 38400 bps. This function set the baudrate at 38400bps if necessary.
Parameters
armPointer to your ARM structure.
enabletrue to enable Wake Up Uart false to disable.
Returns
Error available:
  • ARMN8_ERR_NONE If successful.
  • ARMN8_ERR_WOR_ENABLE If you went disable the Wake Up Uart (enable = false) and the case here the ARMPORT_WITH_nSLEEP is not define and the WOR is enable in ARMN8_WOR_CS or ARMN8_WOR_PQT.
Note
The Wake Up Uart can be disable by armN8SetWorMode().
You need to call armN8UpdateConfig() for update the parameters in you ARM.
After call armN8UpdateConfig() functions, the ARM go to sleep:
  • If the ARMPORT_WITH_nSLEEP is not defined.
  • If ARMPORT_WITH_nSLEEP is defined and sleep is enable (see armN8Sleep()).

Enable/Disable and configure the wake on radio mode.

Warning
If ARMPORT_WITH_nSLEEP is not defined and the mode is ARMN8_WOR_CS or ARMN8_WOR_PQT this function enable the wake up on uart (see armN8EnableWakeUpUart()) and change the serial baudrate if necessary. In other mode (and if ARMPORT_WITH_nSLEEP is not defined) this function disable the wake up on uart.
If you have the LBT mode enable (in ARMN8_LBTAFA_LBT or ARMN8_LBTAFA_LBTAFA see armN8SetLbtAfaMode()) and the mode is ARMN8_WOR_CS the rssiLevel is the same register in ARM, so if change this value with this function, the rssiLevel value of the LBT is also modified.
Parameters
armPointer to your ARM structure.
modeThe value available:
  • ARMN8_WOR_DISABLE To disable the wake on radio mode. The other parameters is ignored.
  • ARMN8_WOR_LP To enable the long preamble, necessary for the transmitter to wake up the receptor. The parameters rssiLevel and filterLongPreamble is ignored.
  • ARMN8_WOR_CS To enable the wake on radio in 'Carrier Sense' (choose this mode for speed).
  • ARMN8_WOR_PQT To enable the wake on radio in 'Preamble Quality Threshold' (choose this mode for quality reception). The parameter rssiLevel is ignored.
periodTimeThe period time in milliseconds.
  • The maximal value possible is 900.
  • The function, return the error ARMN8_ERR_PARAM_OUT_OF_RANGE if this value is out of rang.
  • If the mode is ARMN8_WOR_CS or ARMN8_WOR_PQT the periodTime is the time between two wake up. If the condition of mode is check, the ARM keep awaken to receive the frame. This value must be same of emitter (mode in ARMN8_WOR_LP).
  • If the mode is ARMN8_WOR_LP the periodTime is the time of emission long preamble. It is necessary to emit the loge preamble for considerate the wake-up period of the receiver. This value must be same of receptor (mode in ARMN8_WOR_CS or ARMN8_WOR_PQT).
postTimeThe time in milliseconds where the WOR is disable.
  • The maximal value possible is 2550.
  • The function, return the error ARMN8_ERR_PARAM_OUT_OF_RANGE if this value is out of rang.
  • In the case or mode is:
    • ARMN8_WOR_CS or ARMN8_WOR_PQT, the postTime is the time here the ARM keep awaken after the receive the last frame.
    • ARMN8_WOR_LP after send the first frame with long preamble, the next frames are send without long preamble. After the last frame send and after the postTime the ARM send the next frame with long preamble.
rssiLevelThis parameter in dBm is used by ARMN8_WOR_CS (the default value is -95). In ARMN8_WOR_CS mode the wake up is generate by the rssi level. This Level is configurable by rssiLevel parameter.
filterLongPreambleIn the case where mode is ARMN8_WOR_CS or ARMN8_WOR_PQT, if filterLongPreamble is true the receive frames with long preamble is not push on the serial port.
Returns
Error available:
  • ARMN8_ERR_NONE If successful.
  • ARMN8_ERR_PARAM_OUT_OF_RANGE If one of the parameters is out of rang.
Note
You need to call armN8UpdateConfig() for update the parameters in you ARM.
After call armN8UpdateConfig() functions, the ARM go to sleep:
  • If the ARMPORT_WITH_nSLEEP is not defined.
  • If ARMPORT_WITH_nSLEEP is defined and sleep is enable (see armN8Sleep()).

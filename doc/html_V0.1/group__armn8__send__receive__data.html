Send message to Sigfox network.

Parameters
armPointer to your ARM structure.
bufTxThe buffer of your data at send.
nbyteTxThe number byte to send. Strictly less than or equal to 12.
bufRxThe buffer of receive data, capacity's is must of 8 byte. You can pass NULL is you don't want the acknowledge from Sigfox.
Returns
Error available:
  • ARMN8_ERR_NONE If the data have sent/receive correctly.
  • ARMN8_ERR_SIGFOX_SEND If the data could not be send.
  • ARMN8_ERR_SIGFOX_DATA If nbyte bigger than 12.
  • ARMN8_ERR_PORT_WRITE_READ If can't write or read through the port.
  • ARMN8_ERR_PORT_READ If can't read through the port.
  • ARMN8_ERR_ARM_GO_AT If can't go to AT commend.
  • ARMN8_ERR_ARM_BACK_AT If can't back AT commend.
Warning
If bufRx is not NULL and your ARM don't support the Sigfox in downlink the ARMN8_ERR_SIGFOX_SEND is returned.

Receive data from ARM.

This function is exit when the timeout is achieved from the last receive data or if the nbyte is achieved.

Warning
You can use your method to read data from ARM (for example: in interruption method), but you don't must influence on the behavior of serial port when armN8Init(), armN8Reboot() or armN8UpdateConfig() is call.
Parameters
armPointer to your ARM structure.
bufThe buffer.
nbyteThe size of buffer.
timeoutThe timeout.
Returns
The number of data read. -1 if error from the serial port.

const uint8_t *  buf,

Send data to ARM.

  • In the case or the infinity mode is enable, this function send the data.
  • In the case or the infinity mode is disable, the packet mode is enable, this function send the data to the ARM, 120 byte by 120 byte and wait the necessary time to send the data on the radio.
Warning
You can use your method to write data to ARM (for example: in interruption method), but you don't must influence on the behavior of serial port when armN8Init(), armN8Reboot() or armN8UpdateConfig() is call.
